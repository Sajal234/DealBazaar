# DealBazaar

DealBazaar is a hyperlocal marketplace that connects users with nearby offline stores offering real-time deals.

##  Problem
Local shops don’t have visibility online, and users struggle to find the best deals nearby.

##  Solution
DealBazaar enables verified stores to list limited-time deals and helps users quickly discover and contact them.

---

##  Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MongoDB

---

##  Features (Phase 1 - MVP)

###  User
- Search deals (e.g., "iPhone 15")
- View nearby offers
- Call or WhatsApp stores directly

###  Store
- Apply for store listing
- Add deals (price, images, description)
- Manage active deals

###  Admin
- Approve/reject stores
- Approve/reject deals
- Remove fake or outdated listings

---

##  Core Flow

1. User signs up
2. User applies to register a store
3. Admin verifies store
4. Store posts a deal
5. Admin approves deal
6. Deal goes live
7. Deal auto-expires after 48–72 hours

---

##  Trust System

- Verified store badge
- Store ratings
- Deal expiry system
- Admin moderation
